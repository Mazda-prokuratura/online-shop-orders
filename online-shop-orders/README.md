# Информационная система учёта заказов интернет-магазина

Курсовой проект №29 по дисциплине «Технология разработки программного обеспечения».

## Технологии
- Python 3.12
- Flask
- PostgreSQL
- HTML/CSS (пастельный дизайн)

## Функции
- Учёт клиентов и товаров
- Создание, просмотр, изменение и удаление заказов
- Контроль остатков на складе
- Просмотр состава заказа с расчётом общей суммы

## Установка
1. Создайте БД `shop_db` в PostgreSQL
2. Выполните `db/init.sql`
3. Настройте `.env`
4. Установите зависимости: `pip install -r requirements.txt`
5. Запустите: `python app.py`
